# HTTPS

非对称加密 RSA  -> 对称加密 -> 机构认证

1. 安全大 证书
2.  http 是明文传输 ，https 通过了ssl/tls 加密
3. RSA 公钥加密 私钥解密 速度比较慢


HTTPS 协议的主要功能基本都依赖于 TLS/SSL 协议，TLS/SSL 的功能实现主要依赖于三类基本算法：散列函数 、对称加密和非对称加密，其利用非对称加密实现身份认证和密钥协商，对称加密算法采用协商的密钥对数据加密，基于散列函数验证信息的完整性。
