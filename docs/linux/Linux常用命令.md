5. netstat -nalp t 查看网络端口号的情况。（htop命令）
```
 - n: 列出的带ip地址而不是域名
 - t: tcp/ u: udp
 - p: process 进程的名称
```

```
按 / 可以进行查询。 
可以按大写 F，就会有类似 tail -f 的效果，读取写入文件的最新内容， 按 ctrl+C 停止
n: next
N: pre
```
